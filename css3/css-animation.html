• @keyframes : specify the animation
  • animation : shorthand property
  • animation-delay : when the animation will start
  • animation-direction : alternate cycle
  • animation-duration : time taken by the animation
  • animation-fill-mode : when the animation is not playing
  • animation-iteration-count : number of times the animation
  • animation-play-state : running or paused
  • animation-name : name of @keyframes animation
  • animation-timing-function : speed curve of the animation
